# Visit Counter- web
Built with React in Typescript & Motion for React.
___
## Quick start:
To install the dependencies:
NPM:

```bash
npm install
```

Bun:
```bash
bun install
```

Yarn:

```bash
yarn install
```

___
To start a development server:

NPM:

```bash
npm run dev
```

Bun:
```bash
bun dev
```

Yarn:

```bash
yarn dev
```

To run for production:

NPM:

```bash
npm run start
```

Bun
```bash
bun start
```

Yarn:

```bash
yarn start
```
