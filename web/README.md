# Visit Counter App- web

To install dependencies:

- NPM:

```bash
npm install
```

- Bun:
```bash
bun install
```

- Yarn:

```bash
yarn install
```

___
To start a development server:

- NPM:

```bash
npm run dev
```

- Bun:
```bash
bun dev
```

- Yarn:

```bash
yarn dev
```

To run for production:

- NPM:

```bash
npm run start
```

- Bun
```bash
bun start
```

- Yarn:

```bash
yarn start
```
